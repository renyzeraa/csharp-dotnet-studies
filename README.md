# Estudo de C# e .NET

Repositório de estudos de **C#** e **.NET**, com o objetivo de aprimorar e praticar os fundamentos da linguagem e do ecossistema .NET.

## 🎯 Objetivo

Servir como um diário de aprendizado prático, evoluindo do básico (sintaxe, tipos, console) até tópicos mais avançados do .NET (POO, coleções, LINQ, async/await, etc.).

## 🛠️ Pré-requisitos

- [.NET SDK 10.0](https://dotnet.microsoft.com/download) ou superior
- Um editor à sua escolha: [Visual Studio](https://visualstudio.microsoft.com/), [VS Code](https://code.visualstudio.com/) ou [Rider](https://www.jetbrains.com/rider/)

Verifique a instalação com:

```bash
dotnet --version
```

## 📁 Estrutura

```
c#-studies/
├── Aula00/        # Hello, World! (template padrão de console)
├── Aula01/        # Programa com namespace, classe e método Main explícitos
└── ...            # Novas aulas conforme o progresso dos estudos
```

## 🚀 Como executar

Cada aula é um projeto independente. Para executar uma aula específica:

```bash
cd Aula00
dotnet run
```

Ou aponte diretamente para o projeto desejado:

```bash
dotnet run --project Aula01
```
